## 微信分享配置

所有页面已配置完整的 Open Graph (OG) 标签：
- `og:title` - 页面标题
- `og:description` - 页面描述
- `og:image` - 分享缩略图
- `og:url` - 页面链接
- `og:site_name` - 网站名称
- `og:locale` - 语言区域

## 本地预览

直接用浏览器打开 `index.html` 即可。

## 部署到 GitHub Pages

1. 将所有文件推送到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `Deploy from a branch`
4. Branch 选择 `main` + `/root`
5. 等待 1-2 分钟即可访问

## 技术栈

- 纯静态 HTML + CSS + JavaScript
- 响应式设计（移动端/桌面端适配）
- CSS Grid / Flexbox 布局
